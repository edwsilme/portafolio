Modular web application built with React and Vite, designed to showcase my professional background, projects, and technical skills. This project serves as a practical demonstration of frontend best practices, modularity, and performance optimization.

🔗 **Live Demo:** [edwsilme.github.io/portafolio/](https://edwsilme.github.io/portafolio/)

---

## 🚀 Key Features

* **Internationalization (i18n):** Full multi-language support (Spanish/English) with dynamic content translation and resume download adapted to the selected language.
* **Adaptive UI/UX:** Responsive mobile/desktop interface with native Dark/Light mode support.
* **Performance & Animations:** Smooth transitions implemented using Framer Motion and AOS without compromising load times.

---

## ⚙️ Tech Stack

* **Frontend:** React 18, Vite, JavaScript (ES6+), Tailwind CSS.
* **Key Libraries:** React Router, Framer Motion, AOS, i18next (react-i18next).
* **Deployment & Tools:** Git, GitHub Actions/Pages.

---


## 📂 Project Structure

```text
|- src/
	|- components/    # Modular and reusable UI components
	|- config/        # Data and CV configuration files
	|- hooks/         # Custom hooks (e.g., dark/light theme management)
	|- i18n/          # Language configuration and dictionaries
```

## ⚙️ Installation and Setup

1. **Clone the repository:**
   ```bash
	git clone https://github.com/edwsilme/portafolio.git
   ``` 

2. Navigate to the project directory:
   ```bash
	cd portafolio
   ```

3. Install dependencies:
   ```bash
	npm install
   ```

4. Run in development mode:
   ```bash
	npm run dev
   ```
---

📦 Production Build and Deployment

1. Create a production build:
   ```bash
	npm run build
   ```
   
2. Deploy to GitHub Pages:
   ```bash
	npm run deploy
   ```
